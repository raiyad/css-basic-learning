This is mouse

A computer mouse is an input device that is used with a computer. Moving a mouse along a flat surface can move the cursor to different items on the screen. ... A computer mouse is a handheld hardware input device that controls a cursor in a GUI and can move and select text, icons, files, and folders.

This is keyboard

A computer keyboard is an input device that allows a person to enter letters, numbers, and other symbols (these are called characters in a keyboard) into a computer. ... Using a keyboard to enter lots of data is called typing. A keyboard contains many mechanical switches or push-buttons called "keys".
